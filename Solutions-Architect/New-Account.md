# Creating a new user

## Root user

1. Create account.
2. Add MFA.
3. Enable billing access for IAM users.
4. Add a billing alarm. (use cloudwatch)

## IAM user

1. Set alias for the account.
2. Add new IAM user.
3. Set permissions.
4. Add MFA.
